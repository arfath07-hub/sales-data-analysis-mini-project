## sales-data-analysis-mini-project
Beginner data analysis project using Python and Excel

## Project Overview

This project focuses on analyzing sales data using Python and Excel to extract meaningful business insights. The dataset was stored in an Excel file and processed using the Pandas library.


## Objective

Read sales data from Excel

Perform data cleaning and calculations

Analyze product-wise sales performance

Generate business insights

## Tools & Technologies
Python
Pandas
Excel
Matplotlib (optional)

### Dataset Description
Column	Description
Order_ID	Unique order identifier
Product	Product name
Quantity	Units sold
Price	Price per unit
Sales	Quantity Ã— Price


 ##Steps Performed

Created sales dataset in Excel

Imported Excel file using Pandas

Created a new Sales column

Calculated total sales

Analyzed product-wise revenue

Derived business insights

## Key Insights

Laptop generated the highest revenue

Mobile had the highest quantity sold

Sales increased on later dates

Electronics category dominates overall revenue
